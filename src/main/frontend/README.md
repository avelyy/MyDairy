# frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
### Install Router
```
npm install vue-router@next
```

### Install Axios
```
vue에서 권고하는 Http통신 라이브러리
npm install axios
```
